# Установка и запуск тестов

## Требования

Для запуска тестов необходимо установить **Node.js** (версия 14 или выше).

## Установка Node.js

### Windows

1. Скачайте Node.js с официального сайта: https://nodejs.org/
2. Выберите LTS версию (рекомендуется)
3. Запустите установщик и следуйте инструкциям
4. После установки перезапустите терминал/PowerShell

### Проверка установки

После установки проверьте, что Node.js и npm установлены:

```bash
node --version
npm --version
```

## Установка зависимостей проекта

После установки Node.js выполните в директории проекта:

```bash
cd Dataminer
npm install
```

Это установит все необходимые зависимости:
- Jest (фреймворк для тестирования)
- jsdom (для работы с DOM в тестах)

## Запуск тестов

После установки зависимостей запустите тесты:

```bash
npm test
```

Для запуска тестов в режиме наблюдения (автоматический перезапуск при изменениях):

```bash
npm run test:watch
```

## Альтернативный способ (если Node.js не установлен)

Если установить Node.js нельзя, вы всё равно можете **проверить поведение расширения вручную**:

1. Загрузите расширение в Chrome/Edge (см. `Dataminer/README.md`).
2. Откройте **любую страницу со списком товаров** (например Amazon / Wildberries / Ozon) и пройдите flow `Select → Preview → Export`.

### Оффлайн-страницы из репозитория (для ручной проверки)

В корне workspace есть папка `Test/` с HTML-страницами (пример: `Test/ebay.html`, `Test/wildberris.html`, `Test/amazone_clocks.html`).

Как использовать:

1. Откройте файл в браузере (двойной клик или перетащить в окно браузера).
2. Запустите **Select Elements** и выберите поля кликами.
3. Экспортируйте результат в JSON/CSV и сравните с ожидаемым содержимым.
