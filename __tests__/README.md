# Тесты для Dataminer

## Требования

Для запуска тестов необходимо установить **Node.js** (версия 14 или выше).

Если Node.js не установлен, см. файл `INSTALL.md` для инструкций по установке.

## Установка зависимостей

После установки Node.js выполните:

```bash
npm install
```

## Запуск тестов

```bash
npm test
```

## Описание тестов

### amazon-parsing.test.js

Тесты проверяют корректность парсинга HTML страницы Amazon:

1. **Проверка количества записей**: Тест проверяет, что из HTML извлекается ровно 17 записей товаров.

2. **Проверка заполненности колонок**: Тест проверяет, что все колонки в каждой записи содержат данные (не пустые).

## Структура тестов

Тесты используют:
- **Jest** - фреймворк для тестирования
- **jsdom** - для создания DOM окружения из HTML файла
- **ScrapingService.js** - основная логика парсинга

## Требования

- Node.js версии 14 или выше
- HTML файл `Test/amazon.html` должен существовать
